# 📚 MediQueue - Tutor Booking 

MediQueue is a modern online tutor booking platform where students can easily find experienced tutors, book learning sessions, and manage their bookings securely. The application is built with Next.js, Better Auth, MongoDB, and Express.js.

## 🌐 Live Website

🔗 Live Site: https://mediqueue-ten.vercel.app

## 🚀 Features

- 🔐 Secure authentication using Better Auth (Email/Password & Google Login).
- 👨‍🏫 Tutors can create, update, and manage their tutoring services.
- 🔍 Search tutors by tutor name or subject/category.
- 📅 Students can book available tutoring sessions with date validation.
- 🎯 Automatic seat management that decreases available slots after every successful booking.
- 📌 Users can view and manage all of their booked sessions.
- ✏️ Tutors can edit their profile information anytime.
- 📱 Fully responsive design for Mobile, Tablet, and Desktop devices.
- 🎨 Modern UI built with HeroUI, DaisyUI, and Tailwind CSS.
- ⚡ Protected private routes using JWT authentication.
